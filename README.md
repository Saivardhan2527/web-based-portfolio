# 💼 Personal Portfolio Website

A modern and responsive personal portfolio website built with **Streamlit** to showcase my skills, projects, certifications, and professional journey. This portfolio serves as a central place for recruiters, collaborators, and fellow developers to learn more about my work and explore my technical expertise.

## 🚀 Features

* Clean and modern user interface
* Responsive layout built with Streamlit
* Professional profile and introduction
* Skills and technology stack
* Featured projects with descriptions
* Certifications and achievements
* Contact information and social links
* Custom styling using HTML and CSS

## 🛠️ Tech Stack

* Python
* Streamlit
* HTML
* CSS
* Git & GitHub

## 📂 Project Structure

```text
Portfolio/
│── assets/          # Images and other static files
│── app.py           # Main Streamlit application
│── requirements.txt # Project dependencies
│── README.md        # Project documentation
```

## ▶️ Run Locally

1. Clone the repository.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Start the Streamlit application:

```bash
streamlit run app.py
```

This project was developed to create a professional online presence, demonstrate my technical skills, and showcase my projects in an interactive and visually appealing way. It reflects my passion for software development, data science, and continuous learning.
